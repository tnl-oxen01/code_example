Тестовое задание по вёрстке

Вместо текста по теме, связанной с профессией, я решил разметить здесь описание выполненненного задания, чтобы не создавать для этого отдельный документ. Задание выполнялось с использованием библиотеки JQuery.

"Контрол оценки"

Код "контрола оценки" (включая и связанные с ним таблицы стилей) полснотью раполагается в файле rate.js. Такой подход позволяет легко встраивать данный элемент управления на страницы. Для избежания конфликта имён с уже существующими элементами все перемнные и функции объявлены внутри одного объекта, а к идентификаторам и именам классов элемнетов страницы добавлется префикс, объявленный в переменной prefix. Для изменения иконок необходимо укзать новые ссылки в переменных checkedImg и uncheckedImg. Записав в перемнную callfunc функцию, можно определить действия выполняемые после изменения оценки.Все оценки хранятся в массиве marks, который заполняется функцией getMarks(). Для встраивания данного элемента в страницу необходимо выполнить код как в примере ниже:

Код для добавления "контрола оценки"
Rate.getMarks();
Rate.placeMarkControl($('#rateElement0'),0);
Rate.placeMarkControl($('#rateElement1'),1);
Rate.placeMarkControl($('#rateElement2'),2);

Прогресс-бар

Для прогресс-бара использовался тот же подход, что и для предыдущего элемента, с тем лишь отличием, что описание элемента производится не в объекте а в функции - конструкторе, что позволяет создавать несколько независимых прогресс-баров. Возможно использование двух типов данного элемента: первый отображает прогресс в процентах, второй в единицах от какого-то максимального числа.Ниже приведн пример использования прогресс-баров обоих типов. Обновление значений осуществляет функция markChanged(), описанная в файле index.js, в соответствии с рейтингом, установленным в предыдущем примере.

Код для добавления прогресс-бара
PrBar1 = new ProgBar($('#PrElement1'),'proc');
PrBar2 = new ProgBar($('#PrElement2'),'counter');
Rate.callfunc=markChanged;
Пример отображения прогресс-бара

Кнопки

Кнопки реализованы по тем же принципам, что и прогресс-бар. В примере ниже кнопки позволяют влиять на прогресс-бар, показывающий средний рейтинг, установленный в первом примере. Возможно использование трёх типов кнопок: button - классическая кнопка, trigger - кнопка-переключатель, сохраняющая своё состояние, и radio - радиокнопки, который позволяют выбирать из множества вариантов только один. Важным аспектом использования радиокнопок является то, что каждому объекту кнопки необходимо указать массив radioGroup, в котором содержатся все объекты радиокнопок одной группы.

Код для добавления кнопок
PrBar3 = new ProgBar($('#PrElement3'),'counter');
Btn1 = new MyButton($('#Btn1'),'button','Сбросить рейтинг',resetRate);
Btn2 = new MyButton($('#Btn2'),'trigger','"Заморозить" прогресс-бар');
RBtn[0] = new MyButton($('#RBtn'),'radio','Зелёный',changeColor);
RBtn[1] = new MyButton($('#RBtn'),'radio','Красный',changeColor);
RBtn[2] = new MyButton($('#RBtn'),'radio','Серый',changeColor);
for(var i=0;i<RBtn.length;i++)
    RBtn[i].radioGroup=RBtn;
$('#'+RBtn[0].prefix+'_'+RBtn[0].index).click();
